# file-type-icons
Free file type icons for React

Download these 100% free file format icons, designed specifically for modern web and UI design. These file type icons are available as SVG files, React components, and as a Figma library.

https://www.untitledui.com/resources/file-icons

```
// Single style barrel
import { ZipIcon } from "./file-type-icons/default/jsx";
```

```
// Top-level namespaced barrel
import { defaultStyle, gray, solid } from "./file-type-icons";

<defaultStyle.ZipIcon width={48} />
<gray.ZipIcon width={48} />
<solid.ZipIcon width={48} />
```
